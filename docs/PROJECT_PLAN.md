# Project plan

The authoritative detailed plan is maintained as the companion document
`StreamSense-Serve_详细项目规划.md` in the project workspace. This repository executes it through
the following gates:

1. Evidence schema, event store, adaptive router, and API contract.
2. Deterministic media ingestion with replayable evidence.
3. ASR, audio-event, vision, and VLM adapter implementations.
4. RTX 4090 integration and resource profiling.
5. Baseline, routing, robustness, evidence-quality, and load-test experiments.
6. Reproducible demo, documentation, audit, and public release.

No performance claim may be added to the README or résumé until its raw benchmark artifact,
configuration, software environment, and command are committed or linked from a release.
